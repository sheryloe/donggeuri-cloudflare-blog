Cloudflare-native platform

Public / Admin / API를 나눠서 운영하는 Cloudflare 블로그 플랫폼

이 저장소는 공개 블로그 프런트, 관리자 앱, Worker API를 각각 분리한 재사용용 public template입니다. 실제 도메인과 인증 태그, 분석 설정은 별도 private repo에서 채워 넣는 흐름을 기본으로 둡니다.

저장소 보기

Public Web

Cloudflare Pages로 배포되는 공개 블로그입니다.

Admin

콘텐츠 관리와 세션 기반 접근을 담당하는 관리자 앱입니다.

API Worker

D1, R2, 인증 경계를 담당하는 Worker API입니다.

Data Layer

D1은 글 메타데이터, R2는 미디어 저장 역할을 맡습니다.

Deployment rules

이 프로젝트는 배포 경계를 명확히 가져가는 편이 좋습니다

Public vs Private

template repo에는 엔진과 UI 뼈대만 남기고, 실제 운영값은 private repo로 분리합니다.

Allowlist CORS

Worker는 허용 목록 기반 CORS로 경계를 분명히 해야 합니다.

Separate deploy units

Public/Admin/API를 각기 다른 배포 단위로 유지하는 현재 구조가 장점입니다.

CMS

초안 / 예약 발행

서비스 단계로 가려면 초안 상태와 예약 발행 흐름이 필요합니다.

Security

감사 로그와 역할 분리

관리자 액션 이력과 권한 분리는 운영 안정성을 크게 높입니다.

Public Site

SEO / RSS 마감

검색과 피드까지 완성되면 공개 블로그 플랫폼으로서 가치가 선명해집니다.
