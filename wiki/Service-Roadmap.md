# donggeuri-cloudflare-blog Service Roadmap

## 서비스 목표

Cloudflare 환경에서 Public 블로그와 Admin CMS, API 보안 경계를 분명하게 나눈 서비스형 블로그 플랫폼으로 정리합니다.

## Priority 1

- 관리자 역할과 세션 만료 정책, 감사 로그를 강화합니다.
- 초안, 미리보기, 예약 발행, 미디어 라이브러리를 정리합니다.
- Public 블로그의 SEO, RSS, 사이트맵을 완성합니다.

## Priority 2

- 맞춤 도메인과 운영 가이드를 정리합니다.
- 작성자 경험을 높이는 에디터 보조 기능을 추가합니다.
- 사용자 관점에서 읽히는 소개 페이지와 데모 데이터를 보강합니다.

## Ops

- 배포 경로별 환경변수와 권한 모델을 문서화합니다.
- 관리자 작업 기록을 되짚을 수 있게 로그를 남깁니다.
